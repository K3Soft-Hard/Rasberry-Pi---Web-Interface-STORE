# Plugin STORE for downloading plugins for RaspberryPi Light Web UI
To install plugin you need to dowload not only files in plugin folder, but whole folder with plugin files in it. 
After downloading copy it to Server/plugins/ or open "Pi Transfer", select "Plugin" directory and add folder with plugin.
To enable plugin click on toggle in "Admin Panel". If its not showing restart UI and try again.
